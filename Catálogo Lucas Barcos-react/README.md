# Catálogo Interactivo en React

## Descripción

Aplicación web desarrollada con React que permite visualizar un catálogo interactivo de videojuegos y anime.

La aplicación implementa componentización, manejo de estado, renderizado dinámico y estilos condicionales según los requisitos de la actividad.

## Funcionalidades

* Visualización de catálogo mediante tarjetas.
* Buscador dinámico para filtrar elementos.
* Marcado de ítems como favoritos.
* Identificación visual de elementos destacados.
* Modal con información ampliada del ítem seleccionado.
* Fondo multimedia con video.

## Tecnologías utilizadas

* React
* Vite
* JavaScript 
* CSS

## Estructura del proyecto

* components/: Componentes reutilizables de la aplicación.
* data/: Datos mock utilizados como fuente del catálogo.
* utils/: Funciones auxiliares.
* App.jsx: Componente principal que organiza Header, Main y Footer.

## Ejecución del proyecto

Instalar dependencias:
npm install

Ejecutar servidor de desarrollo:
npm run dev

Abrir en el navegador:
http://localhost:****
